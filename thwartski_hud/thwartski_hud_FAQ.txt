//-------------------------------------------------------------------
//			       1. HOW DO I INSTALL THE HUD?
//-------------------------------------------------------------------

See thwartski_hud__README.txt

//-------------------------------------------------------------------
//			     2. HOW DO I CHANGE MY SCOREBOARD?
//-------------------------------------------------------------------

32-player scoreboard is used by default. 

To use a smaller one, go to \UI\THWARTSKI_SCOREBOARDS\ to find 
the version of ScoreBoard.res you want, and then paste it into the 
\RESOURCE\UI\ folder, overwriting the file. 
	
//-------------------------------------------------------------------
//	     2. WHY IS ONE TEAM CUT OFF FROM MY TOURNAMENT HUD?
//-------------------------------------------------------------------

The 16:9/16:10 tournament hud is used by default, and is too wide for
some smaller resolutions. 

To use the 4:3 tournament hud, go to \UI\THWARTSKI_TOURNAMENT_HUDS\ 
to find the version you want, and then paste it into the \RESOURCE\UI\
folder, overwriting the file. 	

//-------------------------------------------------------------------
//		     3. WHY IS THERE A BUTTON BLOCKING MY MOTD?
//-------------------------------------------------------------------

By default, the continue button is right in the center of the screen, which
blocks the html motd. 

If you have html motds turned on, you can set "cl_hud_minmode 1" in your 
console or autoexec to move that button.  

That's the only thing Thwartski Hud ever uses minmode for. 

//-------------------------------------------------------------------
//	    4. CAN I GET RID OF THOSE ANNOYING MOTD INSTRUCTIONS?
//-------------------------------------------------------------------

If you're tired of seeing the red text reminding you how to move that button
out of the way of the motd, you can replace the file to remove it. 

To use the motd with no instructions, go to \UI\THWARTSKI_MOTDS\ 
to find the version you want, and then paste it into the \RESOURCE\UI\
folder, overwriting the file. 	

//-------------------------------------------------------------------
//	             5. HOW DO I CHANGE MY CROSSHAIRS?
//-------------------------------------------------------------------

Thwartski Hud won't change your crosshairs at all. You can modify your
crosshairs and viewmodels with my other mod, Thwartski Weapon Swapper:

http://code.google.com/p/thwartski-tf2-hud/
